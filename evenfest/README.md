# EVENFEST

**EVENFEST** — сайт для сообщества. Проект выполнен на **FastAPI** с серверными шаблонами **Jinja2** и индивидуальным дизайном.

## Технологический стек
- **Backend**: Python 3.11+, FastAPI
- **Шаблонизация**: Jinja2
- **База данных**: MongoDB (через `motor`)
- **Frontend**: Bootstrap 5 (тема Darkly), Bootstrap Icons, кастомный CSS3
- **Адаптивность**: Мобильная версия с трансформацией боковой панели в нижнюю навигацию
- **Интерактивность**: Изменяемая ширина боковой панели (drag’n’drop) с сохранением в `localStorage`

## Архитектура запуска
Проект поддерживает три режима работы благодаря модульной архитектуре:
1. **В составе экосистемы**: Маршруты монтируются в корневой `main.py`.
2. **Standalone Web**: Запуск локального сервера FastAPI через `python main.py --web`.
3. **Standalone Desktop**: Запуск в виде оконного настольного приложения через `Eel` (просто `python main.py`).
