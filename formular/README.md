# Formular — конвертер документов

**Formular** — веб-приложение для конвертации документов из формата DOCX или HTML в PDF. Проект выполнен на **FastAPI** с асинхронной обработкой файлов и стилизован в готической тематике с оранжевыми акцентами.

## Технологический стек

- **Backend**: Python 3.8+, FastAPI
- **Конвертация**:
  - `python-docx` — чтение DOCX‑файлов
  - `reportlab` — генерация PDF из текста
  - `Playwright` — преобразование HTML в PDF (через headless Chromium)
- **Frontend**: HTML5, CSS3, Bootstrap 5, Bootstrap Icons
- **Интерактивность**: чистый JavaScript (drag-and-drop, загрузка файлов, обратная связь)
- **Статика**: обслуживается через `StaticFiles` FastAPI

Приложение позволяет пользователю перетащить файл в зону загрузки или выбрать его через диалог, указать исходный и целевой форматы (DOCX→PDF или HTML→PDF) и получить готовый PDF для скачивания. Интерфейс адаптирован под мобильные устройства.
