# Snake — неоновая змейка

**Snake** — это классическая игра «Змейка» в современном неоновом исполнении с частицами, анимацией и адаптивным интерфейсом. Проект реализован на **FastAPI** (для раздачи статики) и полностью на **JavaScript** (игровая логика, отрисовка, эффекты).

## Технологический стек

- **Backend**: Python 3.8+, FastAPI (только для обслуживания статических файлов и маршрутизации)
- **Frontend**:
  - Чистый **JavaScript** (ES6) без фреймворков
  - **Canvas API** — отрисовка игрового поля, змейки, еды
  - **Анимация частиц** — создание визуальных эффектов при событиях (поедание еды, повышение уровня, окончание игры)
  - **Bootstrap 5** и **Bootstrap Icons** — элементы интерфейса и кнопки
  - Кастомный **CSS3** — неоновые цвета, анимации свечения, пульсации, адаптивность
- **Управление**: клавиши `W A S D` или стрелки, поддержка русского и английского языков (используются `event.code`), а также кнопки на экране
- **Особенности игры**:
  - Телепортация через границы поля
  - Три типа еды: обычная (зелёная), особенная (голубая), бонусная (жёлтая) — с разной ценностью
  - Повышение уровня (скорости) через каждые 5 очков
  - Индикатор текущего направления движения
  - Эффекты частиц при поедании еды, повышении уровня, окончании игры
  - Полноэкранный режим

Игра запускается в браузере и полностью интерактивна без необходимости в сложном бэкенде.
