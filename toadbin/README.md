# Toadbin — минималистичный сервис для обмена кодом

**Toadbin** — это веб-приложение для быстрого обмена фрагментами кода (аналог Pastebin (но на самом деле katbin)).  
Проект выполнен на **FastAPI** с использованием **MongoDB** для хранения сниппетов и отличается динамической подсветкой синтаксиса и милыми жабками.

## Технологический стек

- **Backend**: Python 3.8+, FastAPI
- **База данных**: MongoDB (драйвер `motor` для асинхронной работы)
- **Шаблонизация**: Jinja2
- **Frontend**:
  - Bootstrap 5, Bootstrap Icons
  - Кастомные шрифты (Garamond, Cinzel, Playfair Display)
  - **Highlight.js** — подсветка синтаксиса в режиме просмотра и редактирования
  - Собственные JavaScript-модули:
    - `codekeys.js` — умные отступы (Tab/Shift+Tab), автозакрытие скобок, обработка Enter с сохранением отступа
    - `save.js` — сохранение по Ctrl+S (независимо от раскладки) с генерацией UUID и редиректом
    - `copy.js` — копирование ID сниппета или всего кода в буфер обмена (с поддержкой мобильных устройств)
    - `random background.js` — выбор случайного MP4-видео из папки `static/backgrounds` и установка его в качестве фона
    - `character counter.js` — счётчик символов
    - `highlight.js` — интеграция highlight.js с текстовой областью (режимы просмотра и редактирования)
- **Интерфейс**: видеофон с тёмным наложением, адаптивный дизайн, кастомизируемый заголовок с инверсией цветов при наведении
- **Особенности**:
  - Режим редактирования (новая вставка) и режим только для чтения (по ссылке)
  - Автоматическая генерация уникального ID (UUID v4)
  - Сохранение сниппетов в MongoDB
  - API для получения списка фонов (`/api/backgrounds`) и существующих ID (`/api/existing-ids`)
