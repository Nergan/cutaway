# Yellow Mirror — веб-зеркало

**Yellow Mirror** — это экспериментальное веб-приложение, выступающее в роли прозрачного прокси/зеркала. Оно загружает удаленные сайты через браузерный движок и транслирует результат прямо на холст клиента.

## Особенности
- **Headless Браузер**: Загрузка сайтов сервером с помощью Playwright.
- **WebRTC/WebSocket Стриминг**: Трансляция изображения пользователю с высокой частотой кадров (через CDP Screencast).
- **Синхронизация ввода**: Перехват и перенаправление мыши, клавиатуры и скролла пользователя на удаленный браузер.
- **Изоляция**: Безопасный просмотр без выполнения чужого кода на машине клиента.

## Архитектура запуска
1. **Meta-App Integration**: Интеграция в общую экосистему.
2. **Standalone Web Server**: Запуск сервера `python main.py --web`.

*(Внимание: Для корректной работы требуется установленный Chromium через Playwright и виртуальный дисплей Xvfb на Linux).*
