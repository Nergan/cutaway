# Yellow Mirror — веб-зеркало

**Yellow Mirror** — это веб-приложение, которое позволяет просматривать внешние сайты через собственный интерфейс, выступая в роли прозрачного зеркала. Все ссылки в загружаемых страницах автоматически переписываются так, чтобы дальнейшая навигация оставалась внутри зеркала. Проект реализован на **FastAPI** и использует асинхронный HTTP-клиент для загрузки ресурсов, сохраняя при этом сессионную привязку к IP-адресу пользователя.

## Технологический стек

- **Backend**: Python 3.8+, FastAPI
- **HTTP-клиент**: `httpx` (асинхронный, с поддержкой keep-alive и переиспользованием соединений на IP)
- **Парсинг и перезапись HTML**: `BeautifulSoup4` + `lxml`
- **Фронтенд**:
  - Чистый **JavaScript** (ES6) с модульной архитектурой
  - **CSS3** с кастомными анимациями, градиентами и параллакс-эффектами
  - Адаптивный интерфейс для мобильных устройств
- **Особенности**:
  - Панель ввода URL, которая сворачивается в тонкую полоску при клике вне её
  - Анимированный многослойный сплэш-скрин с эффектом параллакса при движении мыши
  - Перезапись всех ссылок (`href`, `src`, `action`, CSS `url()`) в загружаемых документах
  - Мониторинг навигации внутри iframe и синхронизация URL-строки браузера (через `postMessage`)
  - Изоляция встроенных страниц через `sandbox`-атрибут iframe
  - Защита от рекурсивных прокси-вызовов и обращений к локальным адресам
  - Поддержка различных методов HTTP (GET, POST, PUT, DELETE и др.)
  - Интеллектуальное определение валидности введённого URL (домены, IPv4, IPv6, localhost)
